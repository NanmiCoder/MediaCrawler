# Cấu Trúc Mã Dự Án

Xem [Kiến Trúc Dự Án](Kien-Truc-Du-An.md) để hiểu chi tiết cấu trúc mã và thiết kế kiến trúc của dự án.

## Cấu Trúc Thư Mục Chính

```
MediaCrawler/
├── main.py                 # Điểm vào chương trình
├── config/                 # Quản lý cấu hình
├── media_platform/         # Triển khai crawler platform
├── store/                  # Lưu trữ dữ liệu
├── database/               # Lớp cơ sở dữ liệu
├── proxy/                  # Quản lý proxy
├── tools/                  # Công cụ tiện ích
└── docs/                   # Tài liệu
```
